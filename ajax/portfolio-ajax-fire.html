Analysing US Land fires between 2013 - 2017

Project Info:

Used 2013-2017 US Land Fires data to create interactive dashboard and visualisations to determine causes of land-fires and number of land-fires in different US states between 2013-17.

Project Details:
  • Technologies:Python, JavaScript, SQL, HTML & CSS
  • Libraries:Pandas, Plotly, Leaflet.js, D3.js, Flask, SQL Alchemy
  • Date:August 23, 2020
